# AI Picture Translator

**AI Picture Translator** is an Android application that leverages artificial intelligence to translate text within images. Capture or select photos, detect text using OCR, and translate it into your preferred language seamlessly.

## 📱 Features
- **Image capture** from camera or gallery
- **Real-time text detection** using **ML Kit** or similar OCR
- **AI-powered translation** supporting multiple languages
- **User-friendly interface** with editable translations
- **Export/share** translated images

## 🛠️ Tech Stack
- **Language**: Kotlin
- **Build System**: Gradle (KTS)
- **Architecture**: MVVM (assumed standard)
- **UI**: Jetpack Compose / XML (update based on your implementation)
- **Dependencies**: AndroidX, ML Kit, Retrofit (common for such apps)

## 📋 Prerequisites
- **Android Studio** (latest version recommended)
- **JDK 17** or higher
- **Android SDK** API level 24+
